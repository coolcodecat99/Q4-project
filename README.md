IF/ELSE STATEMENTS:
- use "elif" = else if
```Python
age = 25

if age < 13:
    print "Child"
elif age >= 13 and age < 20:
    print "Teenager"
elif age >= 20 and age < 65:
    print "Adult"
else:
    print "Senior"

# prints "Adult"
```
